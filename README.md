# Thej Portfolio

A lightweight static personal site for hosting a CV, cover letter, and selected apps.

## Preview locally

Open `index.html` in a browser, or serve this folder with any static-file host.

## Publish

The site is self-contained and can be deployed to GitHub Pages, Netlify, Vercel, or any static hosting service. The CV is stored at `assets/CV-Thej-Siddhotam-2026.pdf`.
